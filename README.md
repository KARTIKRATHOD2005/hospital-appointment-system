# Hospital Appointment Management System

## Tech Stack
- React.js
- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcrypt Password Hashing
- Bootstrap

## Features
- Patient Registration
- Secure Login
- Doctor Management
- Appointment Booking
- Appointment Status Update
- Appointment Deletion
- Admin Dashboard
- Responsive UI

## How to Run

Backend:
cd backend
node server.js

Frontend:
cd frontend
npm run dev
